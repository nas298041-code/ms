
## Usage

生成shellcode
```
ms.exe <输入exe> <输出伪装名称.jpg> <架构: x86或x64>
```

根据架构选则相应加载器加载
```
loader_x64.exe <输出伪装名称.jpg>
```

## Notice

如果使用的过程中出现360 QMV202云引擎报毒，可尝试对加载器替换资源
